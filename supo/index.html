Supo

Para que ningun cobro te tome por sorpresa.

Supo es un gestor de suscripciones simple y privado. Registra cada servicio, te avisa antes de que llegue el cobro y te muestra cuanto estas pagando este mes. Al tocar una suscripcion se abre directo el enlace de gestion de esa plataforma, para que cambies el metodo de pago, actualices la tarjeta o canceles sin vueltas.

Como funciona

01

Agrega tus suscripciones

Registra el nombre, monto y día de cobro de cada servicio para mantener un control claro de tus pagos. Puedes obtener estos datos desde la pagina de tu suscripcion.

02

Recibe recordatorios

Activa notificaciones para recibir un aviso antes del siguiente cobro y evitar pagos inesperados.

03

Asegura las notificaciones

Desactiva el ahorro de batería en la configuración de tu dispositivo para asegurar que las notificaciones lleguen a tiempo.

04

Gestiona tus suscripciones

Desplaza una suscripcion a la derecha para editar y a la izquierda para ocultar. Ademas puedes clickearla para acceder a su sitio web.

05

Administra desde la web

Usa el campo 'Enlace' para acceder a la página de tu suscripción con un solo toque. Si no conoces la dirección, busca la página oficial del servicio, copia su enlace y pégalo en la aplicación. Asegúrate de ingresar un enlace válido para que pueda abrirse correctamente.

Apoyame
